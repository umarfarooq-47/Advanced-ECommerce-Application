# Advanced E-Commerce Application (Nordic Shop) 🛒

An advanced, premium, and fully functional mobile-only E-Commerce web application built under the brand archetype **"Nordic Shop"**. This project is developed as part of the **Android App Development Internship** at **Arch Technologies** to fulfill the Month 1 milestones.

---

## 🎯 Project Overview & Objective
The primary objective of this application is to deliver a lightweight, secure, and modern mobile-first marketplace. The architecture bridges the gap between typical consumer buyer workflows and merchant/vendor decentralized product registration flows, all while enforcing data security and payment processing safeties.

### Key Features Implemented:
* **Premium Minimalist UI:** Built with a "Nordic Shop" aesthetic using clean geometry, slate typography, and emerald green action markers.
* **Interactive Marketplace Grid:** Dynamically maps product catalogs using responsive state layouts.
* **Live Client-Side Search Filters:** Instant character-matching search bar and category filters ("All", "Electronics", "Gear") that re-render components dynamically.
* **Dynamic Shopping Basket Ledger:** A sliding overlay drawer tracker equipped with line-item update metrics (+/- quantity adjusters) and automated pricing subtotals.
* **Merchant Asset Registration System ("Sell Screen"):** A secure form interface allowing instant payload expansion via JavaScript arrays, pushing newly listed items onto the global catalog grid.
* **Secure Checkout Gateway Mask:** A structured payment authorization workflow modal mimicking Luhn validation algorithm structures with trust badges.

---

## 🛠️ Technology Stack
* **Structure & Layout:** HTML5 (Semantic Framework)
* **Styling Engine:** Tailwind CSS via Production CDN Matrix
* **Core Logic & State Management:** Vanilla JavaScript (ES6+ Functional Component Arrays)
